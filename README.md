# Mustache

A MediaWiki extension that allows users with `editsitecss` and `editsitejs` rights to create and edit Mustache templates in a new Mustache namespace and raw HTML pages in a new HTML namespace. Templates and HTML pages can be included in regular wiki pages using the `{{#mustache:}}` and `{{#html:}}` parser functions.

For detailed usage instructions, see the [documentation page on mediawiki.org](https://www.mediawiki.org/wiki/Extension:Mustache).
